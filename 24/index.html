Day 4 - Javascript

Time: 22/2-2016, 13:20 - 15:45 &
Time: 25/2-2016, 13:20 - 15:00

In this class we will be together with the students from the Angular.js elective.

Notice that the time for this class is different than the normal friday classes

Place: L37, 1-A-08 & 1-A-09

Today we will cover the basics of the Javascript language, and we will focus on the EcmaScript 5 standart.

The three semesters of Java Programming you have had so far will make it relative easy for you to learn a new language, including Javascript. And like you would if you where trying to learn C++ og C#, you will spend a little time to get to know the similarities and differences between what you already know and Javascript.

Javascript is often seen as almost similar to Java, and that makes most programmers just start using it before learning it. But as it turns out, it is quite different in many ways and therefore it is good to spend a little time to understand it before you just jump into it.

And this is what we will do today.
