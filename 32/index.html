09.1 Exercise: MongoDB Getting Started

The information you will need for this exercise can be found at this website.

Create a database called store. In this there should be a collection called users, and a collection called products.

Insert
Insert 3 users in the user collection. The 2 below, and one your create yourself (you own name). You can insert these one at a time or all 3 at the same time by wrapping them in an array [ ]

{
        "name" : "Claus",
        "lastName" : "Bové",
        "address" : "Svanemøllevej 56",
        "phone" : "004512345678",
        "email" : "clbo@kea.dk"
}

{
        "name" : "Anna",
        "lastName" : "Messner",
        "address" : "Lygten 16",
        "phone" : "004512345678",
        "email" : "anna@stud.kea.dk"
}


Insert 3 products in the products collection. The 2 below, and one your create yourself. You can insert these one at a time or all 3 at the same time by wrapping them in an array [ ]

{
        "make" : "Burton Menswear London",
        "name" : "Print T-shirt",
        "color" : "white",
        "price" : 16,
        "size" : ["small", "medium", "large", "x-large"],
        "details" : {
                "length" : "standard",
                "fit" : "normal",
                "fabric" : "jersey",
                "Neckline" : "Crew neck",
                "Total length" : "28.0\"  (Size M)"
        }
}

{
        "make" : "ONLY",
        "name" : "ONLNEW - Slim fit jeans",
        "color" : "black olive",
        "price" : 30,
        "size" : ["x-small", "small", "medium", "large", "x-large"],
        "details" : {
                "length" : "long",
                "fit" : "normal",
                "Pockets" : "Back pocket",
                "outer leg length" : "40.0\" (Size S)",
                "inner leg length” :  "31.5\" (Size S)"
        }
}

`

Querying
Select all users, and after that select all products.
Then select the user with your own name, and the a single product of your own choice, by querying on the “_id”

Update
The user with the name “Claus” should have another address.
Change the address to “Lygten 16”

The product ONLY, needs to have the name field changed to “ONLY NEW - Slim fit jeans”

Remove
Remove the user with the name Anna
Remove the product with the make “Burton Menswear London”
