09.2 Exercise: MongoDB 2.0.0 Driver

In the following exercises you should create 4 different applications (4 different javascript files), each containing 1 of the 4 CRUD operations.

You can read about the MongoDB 2.0.0 driver CRUD operations here

Find()
Create an app that reads all users from your “store” database and prints them out in the console.

Insert()
Insert a user into the database. The user's id should be an automatic generated id.

Update()
Update the inner object “details” in the "products" collection so that the length becomes "long" of the "Burton Menswear London" product.

Remove()
Remove one of your users from the store database.

FindOne()
Get the user you inserted earlier in this exercise. You should identify the person from it´s id, and for that you have to use the ObjectID method.
