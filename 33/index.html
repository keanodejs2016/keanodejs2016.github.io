05 Exercise: learnyounode

Learnyounode is an intro to Node.js via a set of self-guided workshops.

It is run through the console and everything is explained along the way.
This workshop covers what you have seen in the Lynda.com tutorials for this class.

Install
In order to install the workshop you should open your console an write:

npm install learnyounode -g

If you get an error try to prepend the command with "sudo" 

sudo npm install learnyounode -g

and type in your password for your computer.

Run
Run the workshop with this command 

learnyounode

Have fun! and see you next friday.
